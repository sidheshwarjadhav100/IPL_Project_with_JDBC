**IPL Squads Management System using MySQL DB**

This project provides a Java-based client interface for managing IPL player information. 
It allows users to perform **CRUD** operations, query player and team data, and retrieve detailed player information through a console-based menu.

**Table of Contents**<br>
The table player contains columns as -<br>
| Serial No (PK) | journey no | player name | runs | wickets | team |


**Features:-**<br>
Insert Player: Insert a new player into a database.
Update Player: Update existing player information.
Delete Player: Delete player from the database.
View Players by Team: View all players of a specific team.
View All Teams: View all teams in the database.
View Player Info: Get detailed information on a specific player.

**Technologies Used**<br>
Java<br>
Java Collections (List, Set)<br>
Console-based UI<br>
JDBC<br>
